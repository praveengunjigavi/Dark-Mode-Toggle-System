# 🌙 Dark Mode Toggle System (Java)

## 📌 Project Overview

The **Dark Mode Toggle System** is a Java-based application that allows users to switch between **Light Mode** and **Dark Mode**, store their preference, and apply the selected theme dynamically.

This project demonstrates core **Object-Oriented Programming (OOP)** concepts such as classes, constructors, encapsulation, and method usage.

---

## 🚀 Features

* 🔁 Toggle between Light and Dark themes
* 💾 Store user theme preference (simulated)
* 🎨 Apply theme dynamically
* 🧱 Clean OOP-based structure
* 🖥️ Console-based interactive system

---

## 🛠️ Technologies Used

* Java (JDK 8 or above)
* OOP Concepts
* Command Line Interface (CLI)

## ▶️ How to Run the Project

### Step 1: Navigate to Project Folder

```bash
cd "C:\Dark Mode Toggle System"
```

### Step 2: Compile Java Files

```bash
javac *.java
```

### Step 3: Run the Application

```bash
java DarkModeApp
```

---

## 🧠 How It Works

1. The application starts in **Light Mode (default)**.
2. Users are presented with menu options:

   * Toggle Theme
   * Apply Theme
   * Save Preference
3. Based on user input:

   * Theme switches between light and dark
   * UI update is simulated using console output
   * Preference is stored (simulated)
